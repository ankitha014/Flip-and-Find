# 🧠 Flip & Find: The Ultimate Memory Match

![Flip & Find Cover](https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/brain.svg)

Welcome to **Flip & Find**, a fast-paced, visually stunning memory match game designed to test your focus and speed. With multiple themes, difficulty levels, and strategic power-ups, it's more than just a matching game—it's a brain-training adventure!

## ✨ Key Features

*   **🎨 Dynamic Themes**: Switch between **Jungle**, **Space**, and **Ocean** environments, each with unique icons and color palettes.
*   **🏆 Difficulty Levels**: 
    *   **Easy**: 4 pairs (60s) – Perfect for a quick warmup.
    *   **Medium**: 6 pairs (90s) – A solid challenge for the focused mind.
    *   **Hard**: 8 pairs (120s) – Only for the memory masters!
*   **⚡ Power-Up System**:
    *   **👁️ Hint**: Briefly reveal a matching pair to get back on track.
    *   **🔀 Shuffle**: Mix up the remaining cards if you're feeling stuck.
    *   **❄️ Freeze**: Stop the clock for 5 seconds to plan your next move.
*   **🔥 Combo Multiplier**: Chain matches together to skyrocket your score with the combo system.
*   **💎 Premium UI/UX**: Built with a sleek **Glassmorphism** design, smooth 3D flip animations, and vibrant particle effects.
*   **🎉 Rewards**: Earn stars based on your performance and celebrate your victory with a confetti explosion!

## 🎮 How to Play

1.  **Select a Theme**: Pick your favorite environment from the main menu.
2.  **Choose Difficulty**: Select your challenge level.
3.  **Flip Cards**: Click on any card to reveal its emoji.
4.  **Find Matches**: Find two identical emojis to clear them from the board.
5.  **Be Fast**: Clear the board before the timer runs out to win!

## 🛠️ Technical Stack

*   **HTML5 & CSS3**: Core structure and advanced animations.
*   **Tailwind CSS**: Modern utility-first styling for a responsive and premium look.
*   **JavaScript (ES6+)**: Game logic, state management, and interactive elements.
*   **Lucide Icons**: Clean and consistent iconography.
*   **Fredoka Font**: A playful yet readable typography choice.

## 🚀 Getting Started

Simply open `memory-game.html` in any modern web browser to start playing! No installation or setup required.
